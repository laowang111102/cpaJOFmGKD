# 书籍搜索系统 python164

本项目是一个基于Python语言的书籍搜索系统，使用Django框架进行开发，后端数据库采用Mysql。系统实现了图书信息与用户账号的增删改查功能，适用于图书信息管理。

## 技术栈

- **主要语言**：Python 3.7+
- **数据库**：Mysql 5.7+
- **前端技术**：Html, Css, Javascript

## 功能模块

- **图书管理**
  - 新增图书
  - 修改图书
  - 搜索图书
  - 删除图书
- **用户账号管理**
  - 新增账号
  - 修改账号
  - 删除账号

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python 3.7 或以上版本
- Mysql 5.7 或以上版本
- 支持Django的Web服务器

## 部署步骤

1. 安装Python 3.7+
2. 安装Mysql 5.7+
3. 安装Django框架
4. 配置数据库连接
5. 运行项目
